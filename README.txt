irclog2html
===========

Converts IRC log files to HTML with pretty colours.

Quick usage for a single log file::

  irclog2html.py --help
  irclog2html.py filename.log

Mass-conversion of logs (one file per day, with YYYYMMDD in the filename)
with next/prev links, with mtime checks, usable from cron::

  logs2html.py directory
